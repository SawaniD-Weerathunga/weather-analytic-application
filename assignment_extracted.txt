Task  Overview  
Build  a  web/API  application  that  does  the  following:  
1.  Reads  city  codes  from  cities.json 2.  Fetches  weather  data  from  OpenWeatherMap  a.  Register  on  OpenWeatherMap to  obtain  an  API  key  b.  Use  the  CityCode  values  to  fetch  weather  data  from  OpenWeatherMap’s  API:   3.  Computes  a  Comfort  Index  Score  for  each  city  (formula  defined  by  you)  4.  Displays  weather  +  score  +  ranking  5.  Provides  server-side  caching  6.  Supports  desktop  +  mobile  layouts  
 
Design  Your  Own  Comfort  Index  Algorithm  
You  must  create  a  custom  “Comfort  Index  Score”  using  at  least  three  parameters ,  such  as:  
●  Temperature  ●  Humidity  ●  Wind  Speed
